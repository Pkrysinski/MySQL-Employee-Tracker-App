Steps to walkthrough video...

-Download NPM packages!
	1. npm init -y
	2. npm i mysql2
	3. npm i inquirer
	4. npm install express

-Log into MySQL, and initialize/use database.
	1. From CLI, navigate to the directory where the downloaded project resides.
	2. Run the following and enter your password...
		mysql -u root -p
	3. Run the db/schema.sql which builds the DB and utililizes the tables...
		source db/schema.sql
